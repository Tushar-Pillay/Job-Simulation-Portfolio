# 📊 AI-Powered Financial Chatbot (BCG GenAI Simulation)

## 🚀 Project Overview  
This project is part of the **BCG GenAI Job Simulation**, where an AI-powered financial chatbot is developed to analyze corporate financial data from 10-K reports.  

The chatbot provides **interactive financial insights** for companies like Microsoft, Apple, and Tesla using structured financial data and rule-based logic.

---

## 🎯 Objectives  
- Extract and analyze financial data from 10-K reports  
- Identify key financial trends (revenue, profit, growth, cash flow)  
- Build a rule-based chatbot to deliver financial insights  
- Simulate real-world AI applications in financial consulting
